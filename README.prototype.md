# 🧭 Tourist AI Prototype

An AI-powered tourist assistant prototype that recognizes landmarks and objects through a webcam, retrieves information from Wikipedia, and speaks descriptions aloud. The project combines computer vision, natural language processing, and text-to-speech to create an interactive travel companion.

---

## 🚀 Features

- 📷 Real-time object recognition using a webcam
- 🧠 Image understanding with OpenAI CLIP
- 🌍 Retrieves information from Wikipedia
- 🔊 Reads descriptions aloud using text-to-speech
- ⚡ Displays predictions with confidence scores
- 🖥️ Simple desktop application built with Python

---

## 🛠️ Technologies Used

- Python 3
- OpenCV
- PyTorch
- Hugging Face Transformers (CLIP)
- Pillow (PIL)
- NumPy
- Requests
- Wikipedia API
- gTTS (Google Text-to-Speech)


```bash

```

3. Install the required packages

```bash

```

---

## ▶️ Running the Project

Run the application with

```bash
python main.py
```

The application will open your webcam and begin recognizing landmarks or objects. When a recognized location is detected, it retrieves a short description from Wikipedia and reads it aloud.
